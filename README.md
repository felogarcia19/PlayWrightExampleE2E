# Automation Testing Portfolio 🚀

A comprehensive collection of automation testing examples showcasing expertise across multiple testing frameworks and tools.

## 📋 Project Overview

This repository contains professional examples of:
- **End-to-End (E2E) Testing** - Playwright
- **Performance & Load Testing** - K6
- **API Testing** - REST/GraphQL automation
- **Integration Testing** - Multi-layer test examples

## 🚀 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Git

### Clone the Repository
```bash
git clone https://github.com/felogarcia19/PlayWrightExampleE2E.git
cd PlayWrightExampleE2E
```

### Navigate to Specific Examples
```bash
# For Playwright tests
cd playwright-tests

# For K6 tests
cd k6-performance

# For API tests
cd api-testing
```
